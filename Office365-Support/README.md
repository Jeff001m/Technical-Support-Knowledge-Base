# Microsoft 365 Support

## Outlook Not Sending Emails

### Possible Causes
- Internet connectivity issues
- Mailbox full
- Incorrect account settings

### Resolution
- Verify internet connection.
- Restart Outlook.
- Repair the Outlook profile.
- Check mailbox storage.
- Reconfigure the email account.

---

## OneDrive Sync Issues

### Resolution
- Restart OneDrive.
- Check available storage.
- Sign out and sign back in.
- Update the OneDrive application.

---

## Microsoft Teams

### Common Problems
- Unable to sign in
- Camera not detected
- Microphone not working
- Poor call quality

### Resolution
- Clear Teams cache.
- Check device permissions.
- Update Teams.
